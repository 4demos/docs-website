The Cordova Plugin creates custom events for JavaScript errors and reports them to New Relic. To manually track JavaScript errors, you can call `  NewRelic.recordError(error)` as follows:

```js

  NewRelic.recordError(error);

```
In the UI, you can track these JavaScript error events with a custom dashboard.

To create a custom dashboard:
1. Go to [one.newrelic.com](https://one.newrelic.com/all-capabilities).
2. Click **Query builder**.
3. Run this query:

  ```sql
  SELECT * FROM `JS Errors`
  ```
4. Click **Add to dashboard**.
