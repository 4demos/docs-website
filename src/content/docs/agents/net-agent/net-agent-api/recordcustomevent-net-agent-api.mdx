<div>
  <h2>Syntax</h2>
  <pre>
    NewRelic.Api.Agent.NewRelic.RecordCustomEvent(string eventType,
    IEnumerable&lt;string, object&gt; attributeValues)
  </pre>
  Records a custom event with the given name and attributes.
  <h2>Requirements</h2>
  <p>Agent version 4.6.29.0 or higher.</p>
  <p>Compatible with all app types.</p>
  <h2>Description</h2>
  <p>
    Records a{' '}
    <a href="/docs/data-analysis/metrics/analyze-your-metrics/data-collection-metric-timeslice-event-data#event-data">
      custom event
    </a>{' '}
    with the given name and attributes, which you can query in the{' '}
    <a href="/docs/query-your-data/explore-query-data/query-builder/introduction-query-builder">
      query builder
    </a>
    . To verify if an event is being recorded correctly, look for the data in{' '}
    <a href="/docs/query-your-data/explore-query-data/dashboards/introduction-new-relic-one-dashboards">
      dashboards
    </a>
    .
  </p>
  <p>
    For related API calls, see the{' '}
    <a href="/docs/agents/net-agent/api-guides/guide-using-net-agent-api#custom-data">
      .NET agent API guide
    </a>
    .
  </p>
  <div className="callout-important">
    <ul>
      {' '}
      <li>
        Sending a lot of events can increase the memory overhead of the agent.
      </li>{' '}
      <li>
        Additionally, posts greater than 1MB in size will not be recorded
        regardless of the maximum number of events.
      </li>{' '}
      <li>Custom Events are limited to 64-attributes.</li>{' '}
      <li>
        For more information about how custom attribute values are processed,
        see the{' '}
        <a href="/docs/agents/net-agent/attributes/custom-attributes">
          custom attributes
        </a>{' '}
        guide.
      </li>
    </ul>
  </div>
