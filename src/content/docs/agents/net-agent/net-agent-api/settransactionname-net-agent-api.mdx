<div>
  <h2>Syntax</h2>
  <pre>
    NewRelic.Api.Agent.NewRelic.SetTransactionName(<em>string</em> $category,{' '}
    <em>string</em> $name)
  </pre>
  Sets the name of the current transaction.<h2>Requirements</h2>
  <p>Compatible with all agent versions.</p>
  <p>
    Must be called inside a{' '}
    <a href="/docs/accounts-partnerships/education/getting-started-new-relic/glossary#transaction">
      transaction
    </a>
    .
  </p>
  <h2>Description</h2>
  <p>
    Sets the name of the current transaction. Before you use this call, ensure
    you understand the implications of{' '}
    <a href="/docs/agents/manage-apm-agents/troubleshooting/metric-grouping-issues">
      metric grouping issues
    </a>
    .
  </p>
  <p>
    If you use this call multiple times within the same transaction, each call
    overwrites the previous call and the last call sets the name.
  </p>
  <div className="callout-important">
    <p>
      Do not use brackets{' '}
      <code>
        [<var>suffix</var>]
      </code>{' '}
      at the end of your transaction name. New Relic automatically strips
      brackets from the name. Instead, use parentheses{' '}
      <code>
        (<var>suffix</var>)
      </code>{' '}
      or other symbols if needed.
    </p>
  </div>
  <p>
    Unique values like URLs, page titles, hex values, session IDs, and uniquely
    identifiable values should not be used in naming your transactions. Instead,
    add that data to the transaction as a custom parameter with the{' '}
    <a href="https://docs.newrelic.com/docs/agents/net-agent/net-agent-api/add-custom-parameter">
      <code> AddCustomParameter()</code>{' '}
    </a>
    call.
  </p>
  <div className="callout-important">
    <p>
      Do not create more than 1000 unique transaction names (for example, avoid
      naming by URL if possible). This will make your charts less useful, and
      you may run into limits New Relic sets on the number of unique transaction
      names per account. It also can slow down the performance of your
      application.
    </p>
  </div>
  <h2>Parameters</h2>
  <table>
    <thead>
      <tr>
        <th width="25%">Parameter</th>
        <th>Description</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <td>
          <p>
            <code>$category</code>
          </p>
          <p>
            <em>string</em>
          </p>
        </td>
        <td>
          Required. The category of this transaction, which you can use to
          distinguish different types of transactions. Defaults to{' '}
          <strong>Custom</strong>. Only the first 255 characters are retained.
        </td>
      </tr>
      <tr>
        <td>
          <p>
            <code>$name</code>
          </p>
          <p>
            <em>string</em>
          </p>
        </td>
        <td>
          Required. The name of the transaction. Only the first 255 characters
          are retained.
        </td>
      </tr>
    </tbody>
  </table>
  <h2>Examples</h2>
  <pre>
    NewRelic.Api.Agent.NewRelic.SetTransactionName("Other", "MyTransaction");
    {'\n'}
  </pre>
</div>
