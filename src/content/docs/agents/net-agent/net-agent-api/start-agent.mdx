<div>
  <h2>Syntax</h2>
  <pre>NewRelic.Api.Agent.NewRelic.StartAgent()</pre>Start the agent if it
  hasn't already started. Usually unnecessary. <h2>Requirements</h2>
  <p>Agent version 5.0.136.0 or higher.</p>
  <p>Compatible with all app types.</p>
  <h2>Description</h2>
  <p>
    Starts the agent if it hasn't already been started. This call is usually
    unnecessary, since the agent starts automatically when it hits an
    instrumented method unless you disable{' '}
    <a href="/docs/agents/net-agent/installation-configuration/net-agent-configuration#service-autoStart">
      <code>autoStart</code>
    </a>
    . If you use{' '}
    <a href="/docs/agents/net-agent/net-agent-api/setapplicationname-net-agent">
      <code>SetApplicationName()</code>
    </a>
    , ensure you set the app name <b>before</b> you start the agent.
  </p>
  <div className="callout-tip">
    <p>
      This method starts the agent asynchronously (that is, it won't block app
      startup) unless you enable{' '}
      <a href="/docs/agents/net-agent/installation-configuration/net-agent-configuration#service-syncStartup">
        <code>syncStartup</code>
      </a>{' '}
      or{' '}
      <a href="/docs/agents/net-agent/installation-configuration/net-agent-configuration#service-sendDataOnExit">
        <code>sendDataOnExit</code>
      </a>
      .
    </p>
  </div>{' '}
  <h2>Examples</h2>
  <pre>NewRelic.Api.Agent.NewRelic.StartAgent();</pre>
</div>
