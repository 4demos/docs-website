<div>
  <h2>Syntax</h2>
  <pre>NewRelic.Api.Agent.NewRelic.IgnoreApdex()</pre>Ignore the current
  transaction when calculating Apdex.<h2>Requirements</h2>
  <p>Compatible with all agent versions.</p>
  <h2>Description</h2>
  <p>
    Ignores the current transaction when calculating your
    <a href="/docs/apm/new-relic-apm/apdex/apdex-measuring-user-satisfaction">
      {' '}
      Apdex score
    </a>
    . This is useful when you have either very short or very long transactions
    (such as file downloads) that can skew your Apdex score.
  </p>{' '}
  <h2>Examples</h2>
  <pre>NewRelic.Api.Agent.NewRelic.IgnoreApdex();{'\n'}</pre>
</div>
