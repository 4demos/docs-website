## Syntax

```
NewRelic.Api.Agent.NewRelic.SetApplicationName(string $name[, string $name_2, string $name_3])
```

Set the app name for data rollup.

## Requirements

Agent version 5.0.136.0 or higher.

Compatible with all app types.

## Description

Set the application name(s) reported to New Relic. For more information about application naming, see [Name your .NET application](/docs/agents/net-agent/installation-configuration/name-your-net-application).

<div className="callout-important">

Updating the app name forces the agent to restart. The agent discards any unreported data associated with previous app names.

</div>

## Parameters

Parameter

Description

`$name`

_string_

Required. The primary application name.

`$name_2`

`$name_3`

_string_

Optional. Second and third names for app rollup. For more information, see [Use multiple names for an app](/docs/agents/manage-apm-agents/app-naming/use-multiple-names-app).

## Examples

```
NewRelic.Api.Agent.NewRelic.SetApplicationName("AppName1", "AppName2");
```
