<div>
  <h2>Syntax</h2>
  <pre>
    NewRelic.Api.Agent.NewRelic.SetApplicationName(<em>string</em> $name[,{' '}
    <em>string</em> $name_2, <em>string</em> $name_3])
  </pre>
  Set the app name for data rollup. <h2>Requirements</h2>
  <p>Agent version 5.0.136.0 or higher.</p>
  <p>Compatible with all app types.</p>
  <h2>Description</h2>
  <p>
    Set the application name(s) reported to New Relic. For more information
    about application naming, see{' '}
    <a href="/docs/agents/net-agent/installation-configuration/name-your-net-application">
      Name your .NET application
    </a>
    .
  </p>
  <div className="callout-important">
    <p>
      Updating the app name forces the agent to restart. The agent discards any
      unreported data associated with previous app names.
    </p>
  </div>

      </tr>

        <td>

  </pre>
