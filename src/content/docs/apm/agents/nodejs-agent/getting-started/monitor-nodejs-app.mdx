## Choose your Node.js agent install method

<CollapserGroup>
  <Collapser
        id="package-manager"
        title="With a package manager"
    >
  To install with a package manager, follow [these steps](https://one.newrelic.com/marketplace?account=3501165&state=773dd1ee-a7a0-1450-0a0e-3daf0035c807) in our guided install. This method has you install the agent with `apm` in the command line. 
  </Collapser>
  <Collapser
        id="docker"
        title="In Docker containers"
    >
    If your apps are deployed in Docker containers, you'll need to alter your existing Dockerfile so it's compatible with our Node.js agent. Our "[Install the Node.js agent for Docker](/docs/apm/agents/nodejs-agent/installation-configuration/install-nodejs-agent-docker/)" doc walks you through installing the agent in the command line. You'll need your app's name and your license key to complete this process.  
  </Collapser>
  <Collapser
        id="host"
        title="On a host"
    >
    Our [install page](https://one.newrelic.com/marketplace?account=3501165&state=31668b21-7bb1-8d71-01b4-c5de77b3382b) walks you through how to install the agent on an application managed by PM2. This method installs the agent via the command line using curl, bash, and sudo.    

    If you use a different process manager (or maybe you don't use one at all), then check out our standard [guided install](https://one.newrelic.com/marketplace?account=3501165&state=4475f6ac-b76d-5b68-f00d-16c026bc9b65). 
  </Collapser>
  <Collapser
        id="For AWS Lambda"
        title="aws-lambda"
    >
    To install an agent for your AWS Lambda functions, our [guided steps](https://one.newrelic.com/marketplace?account=3501165&state=c8a6be34-2449-db6e-3e81-dc710a7aae18) will provide a setup script that automatically configures AWS for you. 
    
    If you have a preference for manual installation via the command line, then navigate over to these [guided instructions](https://docs.newrelic.com/docs/serverless-function-monitoring/aws-lambda-monitoring/enable-lambda-monitoring/enable-serverless-monitoring-aws-lambda-legacy/#node) from our docs.  
  </Collapser>
</CollapserGroup>
