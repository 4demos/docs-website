The Plugin Central platform is an open resource for monitoring critical information about your entire stack. You can install and use the publicly available plugins there. You can also develop your own plugin agents, and publish them for public or private access.

Plugins in Plugin Central are not supported with accounts that host data in the [EU region data center](/docs/using-new-relic/welcome-new-relic/getting-started/introduction-eu-region-data-center).

![Plugins in New Relic One: Plugin Central](/sites/default/files/thumbnails/image/plugins-landing-page-prototype071720.png "Plugins in New Relic One: Plugin Central")

To view plugins in Plugin Central, go to **one.newrelic.com > More > Plugins**.
