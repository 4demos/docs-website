### New

*   **AWS IoT** integration is now available. Check [AWS IoT monitoring integration](https://docs.newrelic.com/docs/integrations/amazon-integrations/aws-integrations-list/aws-iot-monitoring-integration) for details.
*   **AWS EMR** integration is now available. Check [AWS EMR monitoring integration](https://docs.newrelic.com/docs/integrations/amazon-integrations/aws-integrations-list/aws-emr-monitoring-integration) for details.
*   **AWS Health** integration is now available. Check [AWS Health monitoring integration](https://docs.newrelic.com/docs/integrations/amazon-integrations/aws-integrations-list/aws-health-monitoring-integration) for details.
*   **AWS CloudTrail** integration is now available. Check [AWS CloudTrail monitoring integration](https://docs.newrelic.com/docs/integrations/amazon-integrations/aws-integrations-list/aws-cloudtrail-monitoring-integration) for details.
*   **Azure Redis Cache** integration is now available. Check [Azure Redis Cache monitoring integration](https://docs.newrelic.com/docs/integrations/microsoft-azure-integrations/azure-integrations-list/azure-redis-cache-monitoring-integration) for details.
