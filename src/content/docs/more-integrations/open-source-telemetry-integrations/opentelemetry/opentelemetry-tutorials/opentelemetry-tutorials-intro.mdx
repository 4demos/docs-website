If you'd like to try out OpenTelemetry instrumentation on a small demo app, we have tutorials to help you experiment.

## Java tutorials [#java]

Check out our [Java tutorials](/docs/more-integrations/open-source-telemetry-integrations/opentelemetry/opentelemetry-tutorials/opentelemetry-tutorial-java) where you can choose from a variety of options:

* Inspect a pre-instrumented app you can quickly spin up to send data into New Relic.
* Use the OpenTelemetry Java agent to automatically instrument a demo app.
* Manually insert instrumentation into a demo app.
