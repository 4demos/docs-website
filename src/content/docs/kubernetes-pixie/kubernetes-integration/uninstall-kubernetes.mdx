The New Relic Kubernetes integration can be easly uninstalled from your platform. 

Ensure you are using the appropriate context in the machine where you are running Helm and `kubectl`. You can check the available contexts with this command:

```shell
kubectl config get-contexts
```

And switch to the desired context using:

```shell
kubectl config use-context <var>CONTEXT_NAME</var>
```

## Uninstall Kubernetes using Helm (#uninstall-helm)
If you used Helm to install the Kubernetes integration, run:

  ```shell
  helm uninstall newrelic-bundle -n newrelic
  ```

  <Callout variant="important">
    The guided install includes `newrelic-bundle` as the default release name.
  </Callout>


## Uninstall Kubernetes using manifest (#uninstall-manifest)
If you installed Kubernetes integration using a manifest, use the same manifest to uninstall it:

    ```shell
    kubectl delete -f newrelic-infrastructure.yaml
    ```
