If you got started with the free version of New Relic and you're ready use all of our powerful browser monitoring features, you can upgrade your agent from Browser+Lite to Browser Pro+SPA. 

To change your browser edition type, go to **[one.newrelic.com](https://one.newrelic.com/all-capabilities) > Browser > (select an app) > Settings > Application settings**.
