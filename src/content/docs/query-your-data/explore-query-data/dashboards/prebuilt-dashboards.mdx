Through the New Relic UI, you can access a collection of commonly pre-built dashboards, to help you instantly visualize your data. These pre-built dashboards enhance quickstart dashboard discovery, speed up the installation process, and get to instantly visualize your data.

Go to **[one.newrelic.com](https://one.newrelic.com/all-capabilities) > Dashboards** to open the dashboards page.

## How to install pre-built dashboards [#use]

To install a pre-built dashboard:

1. From the dashboards page, search for the pre-built dashboard you want to customize.
2. Click the pre-built dashboard to select it.
3. Select the account in which to add the data and dashboard.
4. Follow the instructions you in the **Setup data source** section. You can skip this step.
5. Click **View dashboard** to start visualizing your data. You can customize your pre-built dashboard, see [Manage your dashboard](/docs/query-your-data/explore-query-data/dashboards/manage-your-dashboard/). You can also change the name and settings, add new content and new widgets, refine the charts, etc.


## How to view more templates [#more-templates]

Browse the full dashboard catalog to find dashboard templates for all the available integrations. Click the **View All** link next to the Popular pre-built dashboards title to see and search for all the available pre-built dashboards. Or you can use these buttons <Icon name="fe-edit-2"/> and <Icon name="fe-edit-2"/> to see the most recent edited pre-built dashboards.
