Read about the requirements and tips for integrating our Python agent with apps that use the [ASGI](https://asgi.readthedocs.io/en/latest/) server [Hypercorn](https://github.com/pgjones/hypercorn). To return to the general install instructions, go to [Install the Python agent](/docs/agents/python-agent/installation-configuration/python-agent-installation). 

## Automatic initialization with admin script [#wrapper-script]

If you start your app with `hypercorn path_to_app:my_app` and use our Python agent **version 8.0.0.179** or higher, you can use the recommended [admin script integration method](/docs/agents/python-agent/installation-configuration/python-agent-installation#integration). For example:

```
NEW_RELIC_CONFIG_FILE=<var>path/to/</var>newrelic.ini newrelic-admin run-program hypercorn path_to_app:my_app
```

## Python agent API [#api]

You can use the [Python agent API](/docs/agents/python-agent/python-agent-api) to monitor any ASGI server or framework. To do this, mark the [ASGI application entry point](/docs/agents/python-agent/python-agent-api/asgiapplication-python-agent-api) and set the [transaction names](/docs/agents/python-agent/python-agent-api/settransactionname-python-agent-api).

## Event loop diagnostic support [#event-loops]

Our Python agent supports `asyncio` event loop diagnostics. For more information, see [Python event loop diagnostics](/docs/agents/python-agent/supported-features/python-event-loop-diagnostics).
