## Before you start [#before-start]

* If you haven't already, you'll need to [sign up for New Relic](https://newrelic.com/signup). 
* If your application setup makes use of any of the following, see those docs. Otherwise, keep reading.
  * [AWS Lambda](/docs/serverless-function-monitoring/aws-lambda-monitoring/enable-lambda-monitoring/enable-aws-lambda-monitoring)
  * [Azure Web Apps](/docs/apm/agents/net-agent/azure-installation/install-net-agent-azure-web-apps) 
  * [Nuget](/docs/apm/agents/net-agent/install-guides/install-net-agent-using-nuget)
* Have questions about compatibility? See the applicable doc: [.NET Framework](/docs/apm/agents/net-agent/getting-started/net-agent-compatibility-requirements-net-framework) | [.NET Core](/docs/apm/agents/net-agent/getting-started/net-agent-compatibility-requirements-net-core)
