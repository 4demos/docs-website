---
headingText: Compatibility and requirements
componentType: default
---

- Our integration is compatible with MySQL version 5.6 or higher.
  <Callout variant="important">
    For MySQL v8.0 and higher, we don't support the following metrics: `db.qCacheFreeMemoryBytes`, `db.qCacheHitRatio`, and `db.qCacheNotCachedPerSecond`.
  </Callout>

- We require TLS version 1.2 or higher, with TLS authentication.
- Check out [the Amazon ECS requirements](/docs/integrations/host-integrations/host-integrations-list/monitor-services-running-amazon-ecs).
