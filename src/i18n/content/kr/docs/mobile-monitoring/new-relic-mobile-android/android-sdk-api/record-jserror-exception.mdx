## 통사론

```
NewRelic.recordJSErrorException(StackTrace stackTrace)
```

ReactNative 앱에 대한 JavaScript 오류 기록

## 요구 사항

하이브리드 앱용 에이전트 버전 6.8.0 이상

## 설명

이 방법을 사용하면 React Native 앱에서 JavaScript 오류를 기록할 수 있습니다.

## 매개변수

이 호출은 `StackTrace` 매개변수를 허용합니다.
