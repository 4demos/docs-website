[Microsoft Azure](https://azure.microsoft.com) 애플리케이션 및 서비스를 모니터링할 수 있는 몇 가지 방법이 있습니다.

* Azure의 New Relic: [Azure Native New Relic Service](/docs/infrastructure/microsoft-azure-integrations/get-started/azure-native) 를 사용하면 Azure Marketplace에서 직접 New Relic을 쉽게 배포할 수 있습니다.
* Azure 통합:[Azure 통합](/docs/infrastructure/microsoft-azure-integrations/get-started/activate-azure-integrations) 은 수십 개의 Azure 서비스에서 원격 분석 데이터를 보고합니다.
* .NET APM: [.NET APM 에이전트](/install/dotnet) 를 설치하여 Azure 애플리케이션을 모니터링하고 자세한 원격 측정 데이터를 수신할 수 있습니다.
