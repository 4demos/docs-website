## 構文

```
NewRelic.recordJSErrorException(StackTrace stackTrace)
```

ReactNative アプリの JavaScript エラーを記録する

## 要件

ハイブリッド アプリ用のエージェント バージョン 6.8.0 以降

## 説明

このメソッドを使用すると、React Native アプリから JavaScript エラーを記録できます。

## パラメーター

この呼び出しは`StackTrace`パラメーターを受け入れます。
