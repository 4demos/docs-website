チャートを作成した後は、データを最適に表示するために外観をカスタマイズすることができます。チャートをさまざまな形式で共有したり、新規または既存のダッシュボードにチャートを追加することができます。

## チャートの外観を変える [#change-appearance]

NRQL モードでクエリを実行するか、データ エクスプローラーを使用してデータを指定しながらチャートを表示すると、New Relic はデータを分析し、データに適合する[チャート タイプ](/docs/insights/use-insights-ui/manage-dashboards/insights-chart-types)を適用します。

選択できるグラフの種類にはいくつかのオプションがあります。グラフの種類を変更するには、現在のグラフの右側にある \[**グラフの種類**] メニューを使用します。

## チャートのカスタマイズ [#customize-charts]

データの表示方法を最適化するために最善を尽くしていますが、他のニーズがある場合もあります。グラフの種類によっては、追加のカスタマイズ オプションを利用できます。

<Collapser
  id="custom-date-time"
  title="日付と時刻のフォーマット"
>
  テーブルとビルボードの日付と時刻の形式をカスタマイズします。そのままにするか、データの種類ごとに形式を**数値**または**日付**に変更するかを選択できます。

  データがタイムスタンプの場合は、日付と時刻の表現方法を選択できます。

  </figcaption>

  <img
    title="Data format: select a format"
    alt="Data format: select a format"
    src={queriesnrqlCustomChartDateFormat}
  />

  <figcaption>
    **[one.newrelic.com](https://one.newrelic.com/all-capabilities) > Query your data** : これはクエリの例であり、データ形式を選択する方法です。
  </figcaption>

  数字については、自動整形するかどうかを選択したり、表示する小数点以下の数を選択したりします。

</Collapser>

<Collapser
  id="customize-y-axis"
  title="Y軸のカスタマイズ"
>
  ラインチャートやエリアチャートでは、Y軸の最小値と最大値を設定することで、データが一定の値の範囲内で表示されるように調整できます。

  カスタマイズオプションが選択されていない場合、 [ダッシュボード](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards) は、0 から上の値までの完全な Y 軸にマージンを加えて自動的に表示します。

  折れ線グラフ、面グラフ、ヒストグラムでは、凡例を無効または有効にすることができます。

  <img

  <figcaption>
    **[one.newrelic.com](https://one.newrelic.com/all-capabilities) > Query your data** : これはクエリの例であり、**凡例**オプションを有効にする方法です。
  </figcaption>
</Collapser>

<Collapser
  id="bar-charts-pie-charts-tables"
  title="棒グラフ、円グラフ、表"

  クエリの[ファセット](/docs/query-your-data/explore-query-data/dashboards/filter-new-relic-one-dashboards-facets)数が棒グラフ、円グラフ、またはテーブルで 2,000 を超える場合、**その他のグループ**ファセットは残りのファセットを集計します。このカスタマイズにより、\[**その他のグループ**] オプションを表示または非表示にすることができます。

### より多くのチャート固有の機能 [#more-types]

チャートタイプ別の機能については、 [チャートタイプ](/docs/insights/use-insights-ui/manage-dashboards/chart-types/) をご覧ください。

## チャートの共有と表示オプション [#share-charts]

ほとんどのチャートには、チャートを埋め込むオプション、チャートを画像として取得するオプション、ダッシュボードにチャートを追加するオプションなど、さまざまなオプションがあります。チャートの一般的なオプションについては、 [Basic UI features](/docs/new-relic-one/use-new-relic-one/get-started/new-relic-one-core-ui-components) を参照してください。

## オープンソースのチャートライブラリを使用 [#victory-charts]

[Nerdpacks](https://developer.newrelic.com/explore-docs/custom-viz/)を使用して、独自のカスタム ビジュアライゼーションを作成できます。また、Formidable と提携して、オープンソースのチャート ライブラリを使用し、独自の「勝利チャート」をダッシュボードにすばやく追加できるようにしました。これらの[カスタム ビジュアライゼーション Nerdpacks](https://developer.newrelic.com/build-apps/customize-nerdpack/)について学習するには、この短いビデオをご覧ください (約4分）。
